# React Server-Side-Render

## Development
To start development, follow there steps:
```javascript
$ git clone https://github.com/rherwig/template-react-16-ssr.git
$ cd react-server-side-render
$ npm i // [--registry=https://registry.npm.taobao.org]
$ npm start
```
